# Frontend (Vue 3) - Esqueleto

Objetivo: UI web (móvil/PC) para ver agenda activa, controlar zonas y consultar histórico.

## Próximos pasos
- Inicializar con Vite:
  - `npm create vite@latest frontend -- --template vue`
- Agregar router (opcional) y cliente HTTP
- Pantallas MVP:
  - Dashboard (estado por zona)
  - Agenda (CRUD + vista activa)
  - Histórico (consulta por período)
