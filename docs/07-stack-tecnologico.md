# Stack Tecnológico

## Hardware
- ESP32 NodeMCU (CP2102)
- Relés 4CH TTL
- Sensor capacitivo de humedad de suelo (v2.0)

## Backend
- Java 17+
- Spring Boot
- Cliente MQTT (HiveMQ MQTT Client recomendado)

## Frontend
- Vue.js 3
- PWA (opcional en MVP)
- WebSocket para estados en vivo (recomendado)

## Dev
- VSCode
- GitHub Copilot (Agent)
- GPT-5.1 Codex
