# Roadmap MVP

## Fase 1 — Conectividad + comandos manuales
- ESP32 conecta a HiveMQ (TLS)
- Publicar telemetría
- Recibir comandos ON/OFF por zona

## Fase 2 — Agenda simple + persistencia local
- Agenda por zona
- Persistir agenda en flash
- Ejecutar agenda offline

## Fase 3 — Backend Java
- CRUD de agendas
- Publicación de sync de agenda (versionado)
- Recepción de estados/eventos

## Fase 4 — UI Vue
- Vista de agenda activa
- Control manual por zona
- Histórico por período
